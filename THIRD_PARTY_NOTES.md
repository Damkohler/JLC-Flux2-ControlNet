# Third-party reference notes

The compact model structure was derived from the user-supplied reference implementations for Alibaba VideoX-Fun's Flux.2 ControlNet and the Flux2Fun ComfyUI experiment. The supplied VideoX-Fun source files identify their upstream Diffusers/Black Forest Labs references and Apache-2.0 licensing in their headers.

This foundation implementation does not import either project at runtime, does not install a global monkey patch, and does not bundle model weights. Licensing and attribution should be reviewed again before a public release.
